# 手动批量导入视频 Implementation Plan

状态：执行中（2026-07-15，HTML 原型已验收；正式 AppKit 实现等待用户验收）

当前进度：任务 1–9 已完成。中部拖入、多文件失败隔离、共享目录监听与阶段进度已完成 Debug build，等待用户验收正式应用后进入提交收尾。

- [x] 1. 明确需求与技术设计
  - 固化手动导入、全局视频判重和统一标签空间。
  - 固化已有视频静默维护、确认删除后移除和进程内后台服务。
  - 完成数据模型、迁移、并发、权限和交互设计。
  - _Requirement: 1–13_

- [x] 2. 制作并验收手动导入 HTML 原型
  - 覆盖已有视频、空资料库、导入中、导入结果、重复导入和取消状态。
  - 使用 macOS 原生三栏结构、Menu Bar 导入命令和就地反馈。
  - “文件 > 导入视频…”及 `⇧⌘I` 均可打开文件夹选择。
  - Toolbar 和空状态均不放置导入按钮；空状态显示菜单路径与快捷键。
  - HTML 稿完成后直接交给用户验收，不执行浏览器自动验收或自动化测试。
  - _Requirement: 1–4, 13_

- [x] 3. 对齐 MVP 产品与架构文档
  - 修订单目录资料库、自动加入、完整重扫、FSEvents 和无法访问筛选等旧表述。
  - 更新 README、MVP 验收边界和项目准备文档。
  - 保留手动导入与静默状态维护的明确边界。
  - _Requirement: 1–13_

- [x] 4. 实现 Schema 5–7 与旧数据迁移
  - 新增授权来源、视频位置和导入运行记录。
  - 将现有 Bookmark 和相对路径迁移到新模型。
  - 建立全局稳定身份与回退路径键索引。
  - 平铺旧 Finder 导入标签，并在来源恢复后补全旧位置的路径回退键。
  - 保留视频 ID、标签关系、Finder 映射、媒体信息和缩略图引用。
  - _Requirement: 2, 3, 6, 8, 10_

- [x] 5. 实现来源授权与手动导入服务
  - 在 LibraryAccessCoordinator 中管理多个 Security-Scoped Bookmark 和进程内访问对象。
  - 递归发现视频并分批执行全局身份 Upsert。
  - 只对首次创建的视频同步 Finder 标签和补全媒体信息。
  - 提供新增、已存在、失败数量和取消能力。
  - _Requirement: 1–6, 8, 12, 13_

- [x] 6. 实现已有视频静默状态维护
  - 按实际目录合并 FSEvents 监听并统一防抖协调；同一父目录中的文件级来源共享监听。
  - 在 `.utility` 文件枚举任务和 DatabaseStore actor 中匹配已有身份并更新位置。
  - 忽略未知新文件；任一可访问来源发现失败时，本轮不确认删除。
  - 全部当前可访问来源协调成功且视频不再拥有位置时删除记录。
  - _Requirement: 4, 7, 9–11, 13_

- [x] 7. 适配文件操作、媒体处理和缓存清理
  - 通过视频位置解析打开、Finder 定位和媒体处理 URL。
  - 文件内容更新时重新补全媒体信息和缩略图。
  - 确认删除后异步清理缩略图缓存。
  - _Requirement: 7–10, 12, 13_

- [x] 8. 实现已验收的 AppKit 导入交互
  - 在“文件”菜单提供“导入视频…”并绑定 `⇧⌘I`。
  - 从 Toolbar 和空状态移除导入按钮，空状态显示菜单路径与快捷键提示。
  - 移除“重新扫描资料库”和“无法访问”筛选。
  - 支持把多个视频直接拖入中部区域，使用系统复制/禁止指针反馈能否导入，并使用文件级授权持续访问原视频。
  - 在中部标题下方接入可取消的阶段进度与结果反馈；单个文件失败时继续处理其余文件。
  - 保持移动更新、确认删除和后台维护无打扰。
  - _Requirement: 1, 4, 7, 9, 13_

- [x] 9. 完成工程验证与真实文件验收准备
  - 验证 Schema 迁移、全局幂等、重叠目录、移动、删除和异常隔离。
  - 执行 Debug build 和自动化测试。
  - 使用真实 Finder 标签、父子目录和外置卷准备用户验收步骤。
  - _Requirement: 1–13_

- [ ] 10. 用户验收与提交收尾
  - 交付 Debug 应用供用户验收。
  - 根据验收反馈修正实现并重新验证。
  - 用户明确验收通过后再整理对应提交。
  - _Requirement: 1–13_
