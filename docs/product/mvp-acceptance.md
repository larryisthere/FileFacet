# MVP 验收记录

更新日期：2026-07-16

## 当前结论

MVP 阶段 0–5 的核心工程已经建立。当前实现采用 AppKit-first，数据库、文件授权、媒体处理和文件监听均使用 macOS 系统能力。视频资料库保持只读，应用不写入 Finder 标签。本文记录自动化与工程验证范围，真实窗口和交互体验仍以用户验收结果为准。

## 自动化验证

- 数据库 schema migration、WAL 与外键初始化。
- Schema 4 到 5 多来源迁移、Schema 5 到 6 Finder 导入标签顶级平铺、Schema 6 到 7 路径回退键迁移，以及手动导入 Upsert 和取消导入保留已完成批次。
- 同一系统视频跨来源全局判重；重复导入、父子目录和重叠目录不会创建重复记录。
- FSEvents 静默维护只处理已有视频；移动保留视频 ID 与标签，确认删除后清理视频、关系和缩略图。
- 同卷文件身份重命名/移动关联，并保留视频 ID 与应用标签。
- 10,000 条合成视频首次索引和 5,000 条标签筛选；筛选断言上限为 200 毫秒。
- 1,000 文件目录发现，以及隐藏目录、Package 和符号链接跳过规则。
- Finder 标签首次入库单向导入、来源记录、跨视频去重和顶级平铺。
- 标签防环、同层重名、后代去重计数、批量三态、合并、删除子树和状态恢复。
- 文件名搜索、最近添加、未打标签、父标签递归和多标签 AND。
- 损坏媒体解析失败后保留视频身份和基础记录。
- UUID 缩略图路径隔离，以及按最旧文件优先的缓存容量清理。
- 应用锁默认关闭、启用验证成功/失败、锁定与解锁失败状态。
- 闲置锁定选项默认“永不”并可持久化。

## 静态与构建验证

- Swift 6 complete concurrency 构建。
- `git diff --check`。
- 日志调用只包含匿名事件、数量、布尔设置和错误类型；不包含完整路径、文件名、标签内容或 Bookmark 数据。
- 静态核对确认核心标签操作和主动移出视频共用单步撤销记录，并通过编辑菜单提供 `Command-Z` 入口。
- 静态核对确认视频移出撤回在导入期间保留；同一原视频已重新导入时复用当前记录并恢复原有标签；数据库提交结果不依赖后续界面刷新成功；撤回失效或下次启动时回收无引用缩略图与来源授权。
- 静态核对确认菜单选择与 Finder 拖入共用导入管线；同一批次中的父文件夹覆盖其内部文件和子文件夹；视频失败与无法读取的输入项目使用独立结果口径。
- Debug App bundle 体积测量。
- `script/build_and_run.sh --verify` 启动并确认进程存在。

## 仍需真实 macOS 交互验收

以下项目依赖真实文件、系统对话框、Touch ID、Finder 或窗口交互，自动化测试无法替代：

- 从菜单栏或 `⇧⌘I` 反复导入多个真实文件夹、退出后 Bookmark 恢复，以及撤销权限后的重新授权提示。
- 从 Finder 把多个真实视频、多个文件夹及两者组合拖入中部区域，核对网格保持原样、系统复制/禁止指针、父子重叠来源合并、阶段进度、取消、空文件夹、部分失败继续处理和分类数量反馈。
- 重启后打开直接拖入的视频，并核对文件在原父目录内移动或重命名后的访问恢复与位置更新。
- 从同一父目录大量拖入视频时，核对监听资源不会随文件数量等量增长。
- 真实文件移动、删除、外置卷离线和授权异常时的静默维护表现。
- 真实 Finder 彩色标签、特殊字符标签和外置本地磁盘样本。
- Touch ID 与 Mac 登录密码回退；睡眠、系统锁屏、切换应用预览和各闲置时间档位。
- 通过应用菜单和 `Command-,` 进入主窗口内设置，并通过应用菜单“关于 Video Tag Manager”直接进入“关于”分类；核对原资料库侧栏、视频网格和检查器全部退出，设置分类侧栏与内容独占工作区，返回资料库后恢复原侧栏状态，并核对隐私设置持久化和“关于”中的版本/构建号展示。
- 大量真实缩略图滚动、快速复用、批量拖拽、标签层级拖拽和 Command-Z 体验。
- 双击调用用户当前默认播放器、Finder 定位和复制完整路径。
- 单选和多选视频后，通过右键菜单末项及 `Command-Delete` 移出资料库；核对确认提示、原视频保留、中部撤回按钮、`Command-Z`、来源位置与应用标签恢复，以及移出后重新导入同一原视频再撤回时保持单条记录。
- Release 签名构建的冷启动时间、归档体积、Gatekeeper 与发布签名。

这些项目属于设备和交互验收边界，不影响自动化回归结论；发布前需要在目标 Mac 上逐项执行。
