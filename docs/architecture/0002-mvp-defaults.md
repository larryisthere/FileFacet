# ADR-0002：MVP 默认产品与技术决策

- 状态：已接受
- 日期：2026-07-15

## 已确认决策

- 最低支持 macOS 14 Sonoma。
- 支持内置磁盘和外置本地磁盘。
- 排除 NAS、SMB、WebDAV 和其他网络资料库。
- 使用系统 SQLite3，不引入第三方数据库运行时。
- 数据库与缩略图保存在 App Sandbox 中，MVP 暂不增加额外静态加密。
- 设置中提供“启用应用锁”开关，默认关闭。
- 开启应用锁前先完成一次系统身份验证；启用后，Mac 睡眠和系统锁屏后始终重新验证。
- 应用切换时始终显示隐私遮罩，身份验证开关关闭时不增加验证步骤。
- 闲置锁定按整个 Mac 的无输入时间计算。
- 标签数量包含全部后代标签中的去重视频。
- “最近新增”按首次进入应用索引的时间计算，默认范围 30 天。
- 缩略图缓存默认上限 2GB。
- 核心标签操作支持 `Command-Z`。
- 扫描不跟随符号链接，默认跳过隐藏目录和 macOS Package。
- 默认播放器可能产生自己的最近播放记录，应用不承诺清理外部播放器数据。

## 隐私边界

应用锁启用后用于限制应用入口，并隐藏应用内的视频列表、标签、路径和缩略图。应用锁不加密原始视频，也不为拥有本机账户或磁盘管理权限的人提供完整防护。
