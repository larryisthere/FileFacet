# Security Policy

## Supported versions

Security fixes are applied to the latest published version of FileFacet.

## Reporting a vulnerability

Please report vulnerabilities privately through the [repository's GitHub Security Advisory page](https://github.com/larryisthere/FileFacet/security/advisories/new). Avoid opening a public issue for unpatched vulnerabilities or including private file paths, filenames, tags, bookmarks, or library data in a report.

Include the affected version, macOS version, reproduction steps, and the smallest safe diagnostic sample needed to understand the issue. Reports will be acknowledged as soon as practical.
